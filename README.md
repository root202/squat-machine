# WtF?
A little selenium script (python3) for automated kniebeugen folter (google translate ftw...).

# Setup
- `python 3 -m venv venv`
- `source ./venv/bin/activate`
- `pip install wheel`
- `pip install -r requirements`

# Running
- `python3 app.py --help`
- `enjoy`
